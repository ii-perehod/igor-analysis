5 SEO-оптимизация

Сайт оптимизирован для Яндекса и Google:

  • Title + H1 — согласованы по ключевым словам: «Ведущий мероприятий», «Москва» — поисковики видят релевантность
  • Meta description — описание с ключевыми словами (70-190 символов для Яндекса)
  • Первый абзац — содержит имя, город, опыт, виды мероприятий, оборудование — всё, что ищут клиенты
  • robots.txt — разрешает индексацию сайта, скрывает админку готово
  • sitemap.xml — карта сайта для поисковиков готово
  • Open Graph — красивое превью при отправке ссылки в Telegram, VK, WhatsApp
  • Twitter Card — превью для соцсетей
  • Schema.org — структурированные данные:
    EntertainmentBusiness FAQPage Person AggregateRating hasOfferCatalog areaServed Телефон Адрес Соцсети
  • GEO-оптимизация — видимость в ИИ-поисковиках (Perplexity, ChatGPT, Алиса): готово
    llms.txt Факты для ИИ Цены в Schema 40+ отзывов
  • Canonical URL — для корректной индексации
  • og:image — фото Игоря для превью ссылок
  • Lazy loading — картинки загружаются по мере прокрутки, сайт открывается быстрее → выше в поиске
  • CSS вынесен в файл — страница легче, стили кешируются браузером, выше скорость загрузки

10 Безопасность

Что сделано для защиты сайта, данных клиентов и бота:

HTTPS (SSL) Всё соединение зашифровано — замок в браузере включено
Vercel предоставляет бесплатный SSL-сертификат автоматически
Защита от DDoS Vercel + Cloudflare CDN — встроенная защита от атак включено
Сайт раздаётся из ближайшего к пользователю дата-центра (CDN)
Персональные данные Сайт НЕ хранит данные клиентов безопасно
Форма заявки отправляет данные напрямую в Telegram Игорю — без промежуточных серверов и баз данных. Ничего не сохраняется на сайте.
Куки и трекеры Нет cookies, нет отслеживания чисто
Яндекс.Метрика и VK пиксель добавятся позже — это стандартные инструменты аналитики, не продажа данных
Валидация формы Проверка обязательных полей (имя, телефон) есть
Нельзя отправить пустую заявку. Защита от случайных нажатий.
XSS-защита Данные из формы отправляются как текст, не как код защищено
Невозможно внедрить вредоносный скрипт через форму заявки
Админ-панель Вход по паролю, данные в localStorage защищено
Доступ только с устройства Игоря. Данные не уходят в интернет.
Внешние скрипты Только шрифты Google Fonts минимум
Никаких чужих скриптов, рекламных сетей или сторонних трекеров
Telegram-бот (токен) требует доработки
Сейчас токен бота будет в коде страницы — технически виден. Планируется перенос на серверную функцию Vercel (бесплатно), чтобы токен был скрыт. До переноса: даже при утечке токена злоумышленник сможет только отправлять сообщения в чат, но не читать их и не получить доступ к данным.
Спам-защита формы планируется
Добавим ограничение: не более 1 заявки в минуту с одного устройства. При необходимости — CAPTCHA.

Итого: сайт безопасен для посетителей — не собирает лишних данных, не ставит куки, работает по зашифрованному каналу. Два момента (токен бота и спам-защита) запланированы к доработке.
